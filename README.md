# K-Means Clustering: Анализ стран мира

Проект по кластеризации стран на основе социально-экономических показателей с использованием алгоритма K-Means.

## Описание

Данный проект применяет неконтролируемое обучение (unsupervised learning) для группировки стран по схожим характеристикам. Используется датасет CIA Country Facts, содержащий информацию о населении, экономике, демографии и географии различных стран мира [file:1].


## Основные этапы анализа

### 1. Загрузка и разведочный анализ данных (EDA)
- Импорт данных о 227 странах с 20 признаками
- Анализ распределений и описательная статистика
- Визуализация взаимосвязей между признаками [file:1]

### 2. Предобработка данных
- **Обработка пропущенных значений**:
  - Climate: заполнение средним по регионам
  - Literacy: заполнение средним по регионам
  - Удаление строк с оставшимися пропусками [file:1]
  
- **Feature Engineering**:
  - One-hot encoding для категориальной переменной Region
  - StandardScaler для нормализации числовых признаков 

### 3. Определение оптимального количества кластеров

Используются три метрики [file:1]:
- **Elbow Method (метод локтя)**: анализ Sum of Squared Distances (SSD)
- **Silhouette Score**: оценка качества разделения кластеров (выше = лучше)
- **Davies-Bouldin Index**: оценка компактности кластеров (ниже = лучше)

### 4. Кластеризация
- Обучение модели K-Means с k=5 (рекомендованное значение)
- Дополнительный анализ с k=15 для более детальной сегментации 

### 5. Визуализация результатов
- Графики метрик качества кластеризации
- Географическая карта с распределением кластеров
- Анализ корреляции признаков с принадлежностью к кластерам 

## Основные признаки

- **Демографические**: Population, Pop. Density, Net migration, Birthrate, Deathrate
- **Экономические**: GDP per capita, Agriculture, Industry, Service
- **Социальные**: Literacy, Phones per 1000, Infant mortality
- **Географические**: Area, Coastline, Climate, Arable, Crops
